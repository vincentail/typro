# Typro

一款跨平台（macOS / Linux / Windows）的 Markdown 编辑器，具备 Typora 风格的实时预览体验。

基于 Electron 35 + React 19 + CodeMirror 6 构建，支持数学公式（KaTeX）、代码高亮（Shiki）、表格、任务列表、脚注等完整 Markdown 特性。

## 快速开始

### 启动后界面

应用启动后会显示一份示例文档。主界面分为三个区域：

| 区域 | 说明 |
| --- | --- |
| **侧边栏**（左） | 文档目录 / 最近文件，可通过 `⌘⇧L` 折叠 |
| **编辑器**（中） | 基于 CodeMirror 6 的 Markdown 源码编辑区 |
| **预览**（右） | 实时渲染的 HTML 预览，支持数学、代码高亮 |

### 新建文件

- 菜单：**File → New**
- 快捷键：`⌘N`（macOS）/ `Ctrl+N`（Windows/Linux）

### 打开文件

- 菜单：**File → Open...**
- 快捷键：`⌘O` / `Ctrl+O`
- 支持格式：`.md`、`.markdown`、`.txt`

### 保存文件

- **快速保存**：`⌘S` / `Ctrl+S`（首次保存未命名文件时会弹出另存为对话框）
- **另存为**：`⌘⇧S` / `Ctrl+Shift+S`

标题栏文件名前出现 `•` 表示有未保存的更改。

## 文件操作

### 最近文件

侧边栏切换到 **Files** 标签页，即可看到最多 10 条最近打开的文件记录，点击即可快速打开。

### 拖放图片

将图片文件从 Finder / 文件管理器拖入编辑区，图片会以 Base64 Data URL 形式内联插入：

```markdown
![文件名](data:image/png;base64,...)
```

> **提示**：对于大图片，建议手动编写相对路径引用（如 `![alt](./images/photo.png)`），避免文档体积过大。

### 自动保存

应用不内置自动保存，请养成使用 `⌘S` 的习惯。关闭时若有未保存内容，会弹出确认提示。

## 视图模式

通过标题栏右侧的图标按钮，或 **View** 菜单切换三种视图：

### 分屏模式（Split View）

默认模式。左侧为编辑器，右侧为实时预览。拖拽中间分隔条可调整比例（最小 20%，最大 80%）。

### 源码模式（Source Mode）

隐藏预览区，全屏显示 Markdown 源码。适合需要精确控制格式的场景。

### 预览模式（Preview Mode）

隐藏编辑器，全屏显示渲染结果。适合阅读和检查最终效果。

### 专注模式（Focus Mode）

按 `F8` 切换。隐藏侧边栏和状态栏，让你专注于写作内容。再次按 `F8` 退出。

## Markdown 语法支持

### 基础语法

```markdown
# 一级标题
## 二级标题
### 三级标题

**粗体**  *斜体*  ~~删除线~~  `行内代码`

[链接文字](https://example.com)
![图片描述](./image.png)

> 引用块

---
```

### 列表

```markdown
- 无序列表项
  - 嵌套项

1. 有序列表项
2. 第二项

- [x] 已完成的任务
- [ ] 未完成的任务
```

### 代码块

使用三个反引号包裹，并指定语言名称以启用语法高亮：

````markdown
```python
def hello(name: str) -> str:
    return f"Hello, {name}!"

print(hello("World"))
```
````

支持的语言包括（不限于）：`javascript`、`typescript`、`python`、`rust`、`go`、`java`、`c`、`cpp`、`css`、`html`、`json`、`yaml`、`bash`、`sql`、`php`、`ruby`、`swift`、`kotlin` 等。

### 表格

```markdown
| 列一 | 列二 | 列三 |
| :--- | :---: | ---: |
| 左对齐 | 居中 | 右对齐 |
| 数据 | 数据 | 数据 |
```

或使用快捷键 `⌘⌥T` / `Ctrl+Alt+T` 快速插入模板表格。

### 数学公式（KaTeX）

**行内公式**，用单个 `$` 包裹：

```markdown
质能方程：$E = mc^2$
```

**块级公式**，用 `$$` 包裹：

```markdown
$$
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$
```

**支持的 KaTeX 功能**：分数、根号、积分、求和、矩阵、希腊字母、运算符等完整 LaTeX 数学子集。

### 脚注

```markdown
这里有一个脚注引用[^1]。

[^1]: 这是脚注的内容。
```

### 链接引用

```markdown
[链接文字][ref]

[ref]: https://example.com "可选标题"
```

## 主题切换

通过标题栏右上角的下拉菜单切换主题，实时生效：

| 主题名 | 说明 |
| --- | --- |
| **Light**（默认） | GitHub 风格浅色主题 |
| **Dark** | GitHub 风格深色主题 |
| **Solarized Light** | Solarized 浅色，护眼低对比 |
| **Solarized Dark** | Solarized 深色 |
| **Dracula** | 流行的深紫色主题 |

应用会跟随 macOS 的深浅色系统设置自动切换 Light/Dark 主题。手动选择后将保持该设置。

主题影响范围：
- 编辑器配色（CodeMirror 主题）
- 预览区渲染样式
- 侧边栏、标题栏、状态栏配色
- 代码块语法高亮配色

## 侧边栏

侧边栏包含两个标签页，可通过 `⌘⇧L` / `Ctrl+Shift+L` 整体折叠。

### 目录（Contents）

自动提取文档中的所有标题（H1–H6），生成可点击的目录树：

- 缩进深度反映标题层级
- 当前光标所在段落对应的标题高亮显示
- 点击条目将编辑器滚动并定位到对应标题行

### 最近文件（Files）

显示最近打开的文件列表（最多 10 条）。点击条目直接打开。

## 导出

### 导出为 HTML

菜单 **File → Export as HTML**，生成包含完整样式的独立 HTML 文件，可直接在浏览器中打开或分享。

### 导出为 PDF

菜单 **File → Export as PDF**，使用 Chromium 渲染引擎打印，输出与预览外观一致的 PDF 文件，支持 A4 页面尺寸。

> **注意**：PDF 导出使用屏幕渲染，图片和代码块颜色与预览区保持一致。
